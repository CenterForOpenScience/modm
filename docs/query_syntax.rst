Query Syntax
============





Operators
---------

Equality
++++++++

=========== =============== ====================================
Keyword     Operator        Description
=========== =============== ====================================
eq          equal to
ne          not equal to
=========== =============== ====================================

Comparison
++++++++++

=========== =============== ====================================
Keyword     Operator        Description
=========== =============== ====================================
gt          less than
gte         less than or
            equal to
lt          less than
lte         less than or
            equal to
=========== =============== ====================================

++++++++++

Keyword     Operator        Description
