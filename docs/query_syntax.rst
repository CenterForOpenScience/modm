.. query_syntax:

Query Syntax
============





Operators
---------

Equality
++++++++

=========== =============== =====================================================
Keyword     Operator        Description
=========== =============== =====================================================
eq          equal to        Match a single value, including inside a list field
ne          not equal to
=========== =============== =====================================================

Comparison
++++++++++

=========== =============== ====================================
Keyword     Operator        Description
=========== =============== ====================================
gt          greater than
gte         greater than or
            equal to
lt          less than
lte         less than or
            equal to
=========== =============== ====================================

Membership
++++++++++

=========== =============== ====================================
Keyword     Operator        Description
=========== =============== ====================================
in          in
nin         not in
=========== =============== ====================================

String Comparison
+++++++++++++++++

=========== =============== ====================================
Keyword     Operator        Description
=========== =============== ====================================
startswith  starts with
endswith    ends with
contains    contains
icontains   contains (case
            insensitive)
=========== =============== ====================================
